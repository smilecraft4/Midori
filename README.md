## Description

An infinite canvas painting app

## Features

- Navigation
  - Zoom
  - Rotate
  - Pan
  - Reset
  - Snap
- Brush
- Eraser
- Pen axis
  - Position
  - Pressure
  - Grip
  - Roll
  - Elevation
  - Azimuth
- Select
  - Lasso
  - Polygon
  - Box
- Transform
  - Rotate
  - Scale
  - Skew
  - Morph
- Fill
- Delete
- Copy
- Cut
- Paste
- Liquify
  - Nudge
  - Grow
  - Shrink
- Ruler
  - Line
  - Curve
  - Elipse
  - Perspective
    - 1 point
    - 2 point
    - 3 point
    - 5 point
- Multiple Viewpoint
- Single File
- Tabs
- Customizable brush presets
- Custom pen axis responses curves
- Text Field
- Layers
- Folders
- Layer mask (applies to folder as well)
- Blend modes (applied to folder as well)
  - Alpha blend
  - Multiply
  - Add
  - Screen
  - Overlay
- Layer effect (applies to folder as well)
  - Fill
  - Outline
  - Noise
